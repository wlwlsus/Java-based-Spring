### 아래 과정을 통해 Spring 전반적인 감을 잡도록 합니다.

- 스프링 프로젝트 생성
- 스프링 부트로 웹 서버 실행
- 회원 도메인 개발
- 웹 MVC 개발
- DB연동 - JDBC, JPA, 스프링 데이터 JPA
- Test Case 작성
